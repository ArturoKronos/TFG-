## 5. Valoración Económica y Control de Costes

La implantación del sistema IDS no requiere inversión en infraestructura física adicional, ya que todo se ejecuta sobre hardware existente y máquinas virtuales.

---

### 5.1 Costes de Infraestructura

| Concepto            | Coste |
|--------------------|-------|
| Hardware existente | 0 €   |
| Máquinas virtuales | 0 €   |
| Red virtual        | 0 €   |
| **Total infraestructura** | **0 €** |

Este cálculo se utiliza para justificar el apartado económico del proyecto.
